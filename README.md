# Tetris-90s-Mini-Game
Projeto de um jogo de Tetris completo feito para rodar diretamente no navegador, com uma interface inspirada em um mini console de mão (handheld) que eu tinha nos anos 90. Foi desenvolvido como uma aplicação single-page em um único arquivo (index.html), unindo HTML + CSS (Tailwind e estilos próprios) + JavaScript.
